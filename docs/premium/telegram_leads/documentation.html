Create Leads in Odoo from messages forwarded to Telegram bot¶

Here we’ll make a telegram bot, that receives your message and create a Note in Odoo with the same content. You’ll need:

  • Telegram account to create a telegram bot

  • AWS Account to create AWS Lambda function

  • Odoo with admin access to setup REST API

To get such personal bot, purchase the package from app store. It includes:

  • Core OpenAPI module. You will not need to purchase it twice and any other custom integration will be discounted

  • Prepared API access configuration needed to this integration

  • Zip archive of python scripts to setup AWS lambda. AWS lambda hosting is free for 1 Million requests and 400 000 GB-seconds per month

  • Detailed documentation for deployment

  • Warranted support for the integration

Get the package here: https://apps.odoo.com/apps/modules/10.0/telegram_leads/
